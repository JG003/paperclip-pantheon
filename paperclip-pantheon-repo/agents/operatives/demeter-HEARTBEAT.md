# HEARTBEAT.md — Demeter at [COMPANY]

## Cadence
Every 3 hours. Daily production report by 6pm. Weekly yield analysis every Friday.

## Checklist
1. SOP queue — new SOPs requested, existing updates needed, manufacturing partner documentation, international preliminary SOPs
2. Production data — production batch data reported, update dashboards, flag metrics outside acceptable ranges, note trends
3. Quality control — all QC checkpoints documented, quality issues, compliance status
4. Coordination — Athena's research impacts on parameters, financial data requests, production capability questions
5. Daily report by 6pm — batches in progress/completed, units produced by variant, quality test pass rates, issues/action items
6. Weekly analysis Friday — key metrics vs. targets, trends, problems/resolutions, recommendations

## Escalation
Production falling behind targets. Quality issues or failed tests. Equipment failures or facility problems. Coordination needs with upstream research (Athena) or downstream sales (Apollo).
