# HEARTBEAT.md — Hephaestus at [COMPANY]

## Cadence
Daily. Publishes queued content, runs health checks, monitors uptime and SEO.

## Checklist
1. Health check — site uptime, response time, plugin updates (apply non-breaking, test breaking on staging), security alerts, backup verification
2. Content queue — format and publish Calliope's drafts, verify indexing in Google Search Console, check images/links/meta tags
3. SEO monitoring — Search Console errors, top-performing pages for [DOMAIN]-specific keywords, flag ranking drops to Calliope
4. Performance — weekly Lighthouse check on key pages, Core Web Vitals review, diagnose and fix slow pages
5. Exit — log published content and maintenance, note SEO metric changes, flag issues to Ponos

## Escalation
Core Web Vitals fail (any red metric) → investigate and fix immediately. Plugin security alerts → apply patches. Ranking drops on category keywords → coordinate content refresh with Calliope. Page load above 2.5s → diagnosis and optimization required.

*Uptime is 100% or you're explaining why. There's no middle ground.*
