# Ariadne — HEARTBEAT File

**Agent:** Ariadne
**Role:** SEO & Search Visibility Specialist
**Division:** Market & Brand
**Heartbeat cadence:** Every 6 hours (4x daily)

---

## Heartbeat Rhythm

Ariadne operates on a 6-hour cycle with different focus areas per cycle. SEO is a discipline of consistent, compounding actions — not bursts.

### Cycle 1 — Morning (6:00 AM)
**Focus: Monitor & Respond**
- [ ] Check Google Search Console for new crawl errors, manual actions, or index coverage issues
- [ ] Review overnight ranking changes for tracked keywords (top 20 priority keywords)
- [ ] Check Google Business Profile insights for any anomalies (review spam, listing edits, Q&A)
- [ ] Review new Google reviews — flag any requiring urgent response to Hestia
- [ ] Check for new backlinks gained or lost overnight
- [ ] Scan Argos's overnight intelligence for competitor SEO moves
- [ ] **RED ZONE:** If rankings drop >5 positions on any primary keyword, investigate immediately and escalate to Ponos

### Cycle 2 — Midday (12:00 PM)
**Focus: Build & Optimize**
- [ ] Execute one item from the technical SEO backlog (fix crawl error, add schema, improve page speed, fix redirect chain)
- [ ] Review one piece of content queued for publication — verify on-page SEO (meta title, description, headers, keywords, internal links, image alt text)
- [ ] Submit one SEO content brief to Calliope if content calendar has gaps in keyword coverage
- [ ] Build or verify 2-3 local citations if local SEO is active
- [ ] Update keyword map if new pages were published since last cycle

### Cycle 3 — Afternoon (6:00 PM)
**Focus: Outreach & Authority**
- [ ] Execute one link building action (outreach email, guest post pitch, broken link opportunity, resource page suggestion)
- [ ] Review competitor content published today — identify keyword gaps or content opportunities
- [ ] Check and respond to any Google Business Profile Q&A
- [ ] Coordinate with Aether on any upcoming PR placements — provide link targets and anchor text
- [ ] Update internal linking on 1-2 existing pages to support newly published content

### Cycle 4 — Evening (12:00 AM)
**Focus: Report & Plan**
- [ ] Log all SEO actions taken today in Mnemosyne's knowledge base
- [ ] Update ranking tracker with end-of-day positions
- [ ] Queue tomorrow's priority tasks based on today's findings
- [ ] Flag any items requiring Daedalus or Hephaestus technical implementation
- [ ] Update weekly report draft with today's data

## Weekly Cadence

### Monday — Weekly SEO Report
- [ ] Compile weekly SEO performance report for Ponos:
  - Organic traffic (vs. prior week, vs. prior month)
  - Keyword ranking movements (improved, declined, new rankings)
  - Google Business Profile metrics (views, clicks, calls, direction requests) per location
  - New backlinks acquired
  - Technical issues resolved vs. outstanding
  - Top-performing pages and underperforming pages
  - Competitor ranking changes on shared keywords
- [ ] Deliver report by 10:00 AM

### Tuesday — Technical Audit Day
- [ ] Run automated technical crawl of primary web properties
- [ ] Compare against prior week's crawl: new errors, resolved errors, changes
- [ ] Check Core Web Vitals scores across key pages
- [ ] Verify all sitemaps are current and submitted
- [ ] Test schema markup validation on recently modified pages
- [ ] Submit any technical implementation requests to Daedalus/Hephaestus

### Wednesday — Content SEO Day
- [ ] Review keyword performance: which target keywords are within striking distance (positions 4-20)?
- [ ] Identify content refresh opportunities (pages that ranked well but are declining)
- [ ] Create or update content briefs for Calliope based on keyword gaps
- [ ] Audit internal linking structure for top 10 priority pages
- [ ] Review competitor content published this week — identify angles they covered that the company hasn't

### Thursday — Local SEO Day (if applicable)
- [ ] Audit Google Business Profile(s) for accuracy and completeness
- [ ] Check NAP consistency across top 20 citation sources
- [ ] Review and respond to all new Google reviews (coordinate with Hestia)
- [ ] Check local pack rankings for priority local keywords across target geographies
- [ ] Identify new citation opportunities or directory listings
- [ ] `[COMPANY-SPECIFIC]` Review location-specific landing page performance

### Friday — Link Building & Authority Day
- [ ] Review backlink profile: new links, lost links, competitor link gaps
- [ ] Execute 3-5 link building outreach actions
- [ ] Coordinate with Aether on upcoming PR — ensure link strategy is aligned
- [ ] Research new link building opportunities (industry directories, partner sites, trade associations)
- [ ] Update link building tracker with week's results

### Saturday/Sunday — Passive Monitoring
- [ ] Automated rank tracking runs

## RED ZONE Triggers — Immediate Action Required

These conditions require Ariadne to break cadence and respond immediately:

| Trigger | Action |
|---------|--------|
| Google manual action or penalty notification | Investigate immediately. Escalate to Ponos. Begin remediation plan within 1 hour. |
| Primary keyword drops >5 positions overnight | Investigate cause (algorithm update, technical issue, competitor move). Report findings to Ponos within 1 cycle. |
| Google Business Profile suspended or flagged | Begin reinstatement process immediately. Alert Ponos. |
| Site de-indexed or major crawl error spike (>50% increase) | Coordinate with Daedalus/Hephaestus for immediate technical investigation. |
| Confirmed Google algorithm update | Assess impact on rankings within 24 hours. Report to Ponos with recommended response. |
| Negative SEO attack detected (toxic backlink spam) | Document, begin disavow process, escalate to Ponos. |
| Competitor launches in company's primary local market | Accelerate local SEO activity. Report competitive threat assessment to Ponos. |

### Business Metrics (quarterly tracking)

## Handoff Protocols

### When Ariadne Receives Work
- From **Ponos**: strategic SEO priorities, new market/location expansion, algorithm update response directives
- From **Argos**: competitor intelligence requiring SEO response (new competitor rankings, content launches, link campaigns)
- From **Calliope**: content ready for SEO review before publication
- From **Daedalus/Hephaestus**: technical changes requiring SEO review (new pages, site restructure, migration)

### When Ariadne Hands Off Work
- To **Calliope**: content briefs with keyword targets, search intent, structure requirements, and internal linking targets
- To **Daedalus/Hephaestus**: technical SEO implementation tickets (schema markup, page speed fixes, redirect maps, sitemap updates)
- To **Iris**: image optimization specifications (alt text, dimensions, compression, WebP conversion)
- To **Aether**: link building opportunities tied to PR placements (target URLs, anchor text, link placement guidance)
- To **Hestia**: Google review response priorities and review generation campaign coordination
- To **Mnemosyne**: all SEO decisions, ranking data, audit results, and strategy changes for institutional memory
