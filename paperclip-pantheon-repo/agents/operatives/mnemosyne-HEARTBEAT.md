# HEARTBEAT.md — Mnemosyne at [COMPANY]

## Cadence
Every 6 hours. Knowledge management is continuous infrastructure — if it falls behind, everything degrades.

## Checklist

1. Contact updates — new contacts from agent activity? Interactions logged? Follow-ups overdue?
2. Decision log — decisions made since last heartbeat? Logged with context? Past decisions being revisited?
3. Cross-agent context pushes — which agents need context for active tasks? Push proactively.
4. Lessons capture — completed campaigns/events/experiments/deals? Debrief captured and tagged?
5. Knowledge gaps — any agent without full context? Relationship gaps >30 days? Missing decisions?

## Escalation

Flag to Ponos: same decision reconsidered without original context, critical contact stalled >30 days, past failure lesson unavailable to agent making similar decision.

## Templates

Context push table, relationship dashboard, monthly audit, package maintenance → `mnemosyne-HEARTBEAT-TEMPLATES.md`

*Memory doesn't shout. It whispers at exactly the right moment: "We've been here before. Here's what happened last time."*
